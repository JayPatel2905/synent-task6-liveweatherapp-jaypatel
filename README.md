# Live Weather Application

## Description
A Python-based weather application that fetches real-time weather information using OpenWeatherMap API.

## Features
- Real-time weather details
- Temperature display
- Humidity information
- Wind speed
- Weather condition
- Error handling
- Repeat search functionality

## Technologies Used
- Python
- Requests Library
- Python Dotenv
- OpenWeatherMap API

## Installation

Install required libraries:

pip install requests python-dotenv
